# 活动素材分类规则

每次收到新活动的 branding PDF 后，按以下规则将素材归档到对应目录，生成器才能正确识别和使用。

---

## 分类体系

### 主体素材 Hero  `B4/` + `presets/`

**定义**：画面的视觉锚点，尺寸大、层级高，通常居中摆放，决定整体气质。

| 特征 | 说明 |
|------|------|
| 视觉权重 | 最高，占画面 30–60% |
| 位置 | 固定区域（居中偏上），不走弹性布局 |
| 来源 | B4 图形组合贴纸 / presets 预制角色组 |
| 透明底 | 必须，否则会出现色块 |
| 数量建议 | 每个模板 1 个主体素材 |

**本次春节素材（B4）：**
- `horse-lantern.png` — 马 + 心形灯笼组合

**何时用 B4 vs presets：**
- B4：独立贴纸，有文字/角色/图标的组合，适合局部放置
- presets：整体弧形构图（多角色群像），用作全屏背景层的主视觉

---

### 辅助元素 Decor  `B1/` + `B2/`

**定义**：丰富画面层次的散件，在约束范围内弹性排布，不同种子产生不同版本。

| 特征 | 说明 |
|------|------|
| 视觉权重 | 低，起点缀作用 |
| 位置 | 弹性布局（zone + rotation + scale 范围内随机） |
| 来源 | B1 平面元素（矢量插画）/ B2 3D 元素（立体渲染） |
| 透明底 | 必须 |
| 数量建议 | 每个模板 2–5 个，避免遮挡主文字区 |

**本次春节素材（B2）：**
- `plum-blossom.png` — 梅花枝（适合顶角）
- `pine-tree.png` — 松树盆栽（适合右侧）
- `projector.png` — 放映机（适合 banner 右侧）
- `red-envelope.png` — 红包（适合底角）
- `microphone.png` — 话筒（适合底角 / banner）
- `lantern.png` — 灯笼（适合底角）

**B1 vs B2：**
- B1：扁平插画风格（颜色鲜艳、线条感），适合轻快场景
- B2：3D 立体渲染（有光影），质感更强，适合大促重场景

---

### 背景底纹 Pattern  `B3/`

**定义**：可平铺的底纹图案，叠加在渐变背景上增加细节，本身需带透明通道（透明底），背景色由模板单独配置，两者解耦。

| 特征 | 说明 |
|------|------|
| 视觉权重 | 极低（opacity 5–15%），不抢主视觉 |
| 位置 | 全画面平铺 `background-repeat: repeat` |
| 透明底 | **必须从 .ai 导出透明版本**，PDF 提取版本带背景色仅作参考 |
| 数量建议 | 每个模板最多 1 个底纹 |

**本次春节底纹（B3）：**
- `pattern-tiktok-yellow.png` — TikTok logo 重复底纹（⚠ 当前版本带黄底，需 .ai 导出透明版替换）

---

### 颜色体系 Colors

所有颜色来自 branding PDF 色彩规范页，录入 `src/campaigns/{id}/colors.ts`。

| 变量 | 用途 |
|------|------|
| `primary` | 主色（按钮、标题背景、标签） |
| `secondary` | 辅色（渐变终止色） |
| `yellow` | 点缀色 |
| `warmBg` | 暖底卡片背景色 |
| `gradients` | 背景渐变方案列表（可多选） |

---

## 目录结构规范

```
public/assets/{campaign-id}/
  B1/          平面元素散件（透明底 PNG）
  B2/          3D元素散件（透明底 PNG）← 可从 PDF 自动提取
  B3/          底纹图案（透明底 PNG）← 必须从 .ai 手动导出
  B4/          图形组合贴纸（透明底 PNG）← 可从 PDF 自动提取
  presets/     预制角色群像（透明底 PNG）← 需 .ai 手动导出
```

## 提取方式速查

| 分类 | 能否从 PDF 提取 | 说明 |
|------|--------------|------|
| B2 3D散件 | ✅ 自动提取 | `node scripts/extract-pdf-assets.mjs <pdf> <campaign-id>` |
| B4 组合贴纸 | ✅ 自动提取 | 同上 |
| B1 平面元素 | ❌ 需 .ai 导出 | 矢量图，PDF 中无栅格 |
| B3 底纹（透明） | ❌ 需 .ai 导出 | PDF 版本背景色已合并 |
| presets 角色组 | ❌ 需 .ai 导出 | 复杂构图，需设计师合图后导出 |
